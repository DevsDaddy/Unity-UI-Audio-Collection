# UI Audio Collection for Unity
<img src="https://github.com/DevsDaddy/Unity-UI-Audio-Collection/blob/main/DevsDaddy/Audio%20Libraries/UI/Preview.png?raw=true" alt="UI Audio Collection for Unity" style="width: 100%" /><br/>
I present to you a **large collection of audio UI elements** for your Unity-based games **(200+ clips)**. Here you will find anything **from click and minimize sounds to the sound of opening chests or making purchases** in your game.

**You are only limited by your imagination!**

## How to Use
Just use this effects as single audio-sources or see my <a href="https://github.com/DevsDaddy/UIFramework">UI</a> and/or <a href="https://github.com/DevsDaddy/AudioFramework">Audio Framework</a> or <a href="https://github.com/DevsDaddy/OneUIKit">OneUI Kit</a>.
By default all audio clips compressed using Vorbis 40% in Unity.
